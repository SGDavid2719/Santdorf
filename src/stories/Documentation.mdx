# Colors

Santdorf uses a carefully selected palette to ensure an aesthetically pleasing and accessible interface. Below are the color guidelines:

## Primary Colors

```js
const colors = {
  primary: '#003366',
  error: '#ff6f61',
  success: '#31644f',
  warning: '#eab676',
  background: '#1f1f1f',
  accent: '#ffd700',
};
```

### Usage

- Primary: Used for primary actions and highlights.
- Error: Used for error states.
- Success: Used for success states.
- Warning: Used for warning states.
- Background: Used for the main background color.
- Accent: Used for interactive elements.

# Typography

Typography is a crucial aspect, providing a clear hierarchy and readability.

## Font Family

```
font-family: 'sans-serif', 'serif';
```

## Font Sizes

- Base size: 1em
- Scale ratio: 1.2

### Usage

- xs: calc(1em / calc(10px \* pow(var(scale-ratio), 2)));
- sm: calc(1em / var(scale-ratio));
- md: calc(1em \* var(scale-ratio));
- lg: calc(1em _ calc(10px _ pow(var(scale-ratio), 2)));
- xl: calc(1em _ calc(10px _ pow(var(scale-ratio), 3)));
- xxl: calc(1em _ calc(10px _ pow(var(scale-ratio), 4)));
- xxxl: calc(1em _ calc(10px _ pow(var(scale-ratio), 5)));

# Spacing

Consistent spacing is essential for a clean and organized interface.

## Scale

- Base unit: 1em

### Usage

- xxxs: calc(0.25 \* var(unit));
- xxs: calc(0.375 \* var(unit));
- xs: calc(0.5 \* var(unit));
- sm: calc(0.75 \* var(unit));
- md: calc(1.25 \* var(unit));
- lg: calc(2 \* var(unit));
- xl: calc(3.25 \* var(unit));
- xxl: calc(5.25 \* var(unit));
- xxxl: calc(8.5 \* var(unit));
